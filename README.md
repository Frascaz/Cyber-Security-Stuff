## Getting Started

### Dependencies

* A Linux environment (Kali Linux or Parrot OS recommended).
* Python 3.x installed.
* Security tools: Nmap, Burp Suite, Wireshark, Nitko, John The Ripper.

### Installing

1. Clone the repository to your local machine:
   git clone https://github.com/Frascaz/Cyber-Security-Stuff.git

2. Navigate into the directory:
   cd Cyber-Security-Stuff

### Executing program

1. For Python scripts:
   python3 script_name.py

## Help

Common issues usually involve missing dependencies or networking configurations in virtual environments. Ensure all Python requirements are installed via:
pip install -r requirements.txt
